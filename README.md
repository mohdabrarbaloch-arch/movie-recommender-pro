# 🎬 CineWatch Pro: AI-Powered Movie Streaming Platform

CineWatch Pro is a high-performance web application that combines **Machine Learning** with a seamless **Streaming Experience**. Built using Python and Streamlit, it offers personalized movie exploration with a premium, cinematic user interface.

## 🚀 Key Features
* **Intelligent Exploration:** Browse a vast library of movies with real-time data fetching.
* **Multi-Engine Streaming:** Integrated with 3 different streaming servers (Main, Alternative, and Cloud) to ensure 99% uptime.
* **Dynamic UI Themes:** Supports multiple DaisyUI themes including `dark`, `luxury`, `dracula`, and `coffee`.
* **Fast Content Delivery:** Optimized using Streamlit's caching mechanism for instant poster loading.
* **Responsive Design:** Glassmorphism-based UI that looks great on all desktop screens.

## 🛠 Tech Stack
* **Frontend:** [Streamlit](https://streamlit.io/) (Python-based Web Framework)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [DaisyUI](https://daisyui.com/)
* **Backend Logic:** Python (Pandas, Requests)
* **Data Source:** [TMDB API](https://www.themoviedb.org/documentation/api)
* **Deployment:** Streamlit Community Cloud
